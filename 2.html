Soal no 2

Buatlah sebuah function untuk memverifikasi valid password dengan requirements:

  • Berjumlah minimal delapan karakter
  • Harus terdiri dari huruf kecil dan angka, karakter selain itu salah

Contoh valid password adalah

  • qwert123
  • tgby7654

Clue:

  • isPasswordValid(‘tgby7654’);

return==> true

  • isPasswordValid(‘iuoiuuUU’);

return==> false

Untuk Pengecekan jawaban bisa di cek melalui di console.log pada browser dengan memanggil function 'isPasswordValid(argumen)' beserta parameternya
